# Actividad de práctica integradora creada por Alejandro Marvich

## Restaurante

Un restaurante requiere un sistema que permita hacer un seguimiento de las mesas del lugar y los pedidos realizados por cada mesa.  
Se necesita un registro que contenga los productos y sus detalles como el **precio por unidad** y el **nombre**.  
Además, se requiere una estructura adecuada para manejar las **mesas**, el **gasto total**, los **pedidos** y las **personas en la mesa**.

## Lista de requisitos

- Debe ser posible registrar clientes en una mesa, y agregar pedidos.
- Una vez que los comensales se retiren debe limpiarse la mesa para nuevos clientes
- Llevar conteo de los ingresos totales y mostrarlos
- Se debe poder realizar una búsqueda de una mesa específica y mostrar los datos.
