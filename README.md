Para compilar un archivo de codigo .c debemos utilizar el siguiente comando:
El comando *gcc* significa **GNU Compiler Collection**.
- Compiler: Compilador, encargado de traducir código fuente a código máquina.
- Collection: Colección, porque no es solo un compilador de C, sino un conjunto que incluye compiladores para varios lenguajes (C, C++, Objective-C, Fortran, Ada, Go, etc.).
```
gcc *nombre del archivo .c*
```
Si queremos especificar el nombre de salida del archivo ejecutable utilizamos la opcion "-o".
```
gcc *nombre del archivo .c* -o *nombre de salida*
```
